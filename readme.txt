Please Do the following actions in the terminal to visit the food delivery app
1. npm i
2. npm i json-server
3. npm start (do this step in one terminal and the next step in another terminal)
4. npm run serve-json (do this in another terminal)

Now navigate to http://localhost:3000 to visit the food delivery app and open 
http://localhost:4000/items to keep track of the database and now start shopping 
in the website to see how it works, the key functionalities include adding our 
favourite items to cart and deleting them from cart and placing the order.
